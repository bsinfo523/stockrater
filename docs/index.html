GitHub Einstellungen

Mit einem GitHub Token können manuell hinzugefügte Aktien automatisch in docs/custom_stocks.txt gespeichert werden – ohne GitHub-Website öffnen zu müssen.

Token erstellen:
GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token
Nur repo Berechtigung ankreuzen. Der Token wird nur lokal in deinem Browser gespeichert.
